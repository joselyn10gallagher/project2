Practical Special Effects
Practical Special Effects is an effect that is produced physically without the use of computer technology. The techniques film makers use/d are props, stop motion, paintings, puppets, and make-up. These elements are use to create settings and hands-on objects in sets.

In Jurassic Park, multiple dinosaur puppets were created and used throughout the movie.


Visual Special Effects
Visual effects is an effect where imagery is created with computer technology. Films that use CGI (Computer-Generated Imagery) use techniques such as: shooting scenes with a green screen, then animators create a character, effect, or location where that image will replace the green screen in the final product.

In Jurassic World, many of the dinosaurs were created using CGI.


Special Effects and Visual Effects are both used in the movie making process. However with technology constantly growing, more movies are using Visual Effects. This takes away the significance that Special Effects has and should be used more often.
